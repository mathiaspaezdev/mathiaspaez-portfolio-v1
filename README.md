# Portfolio — Versión 1.0

Este es mi primer portfolio como desarrollador, diseñado y construido desde cero como parte de mi proceso real de aprendizaje.  
No es perfecto: es la versión 1.0, una base mínima pero funcional que iré actualizando a medida que avance en HTML, CSS, JS y buenas prácticas.

Mi objetivo es documentar mi progreso públicamente.

---

##  Sobre esta versión

La V1.0 incluye:

- Estructura inicial del sitio (HTML + CSS + un poco de JS)
- Sección Hero con mensaje principal y video embed
- Newsletter (estructura visual)
- Sección “Lo que estoy haciendo ahora”
- Primer diseño del portfolio profesional
- Sección de contacto con formulario básico
- Identidad visual inicial basada en negro + cian + amarillo
- Diseño inspirado en una estética minimalista y documental

Esta versión no está optimizada ni pulida. Es intencionalmente básica, porque representa exactamente dónde estoy hoy como FullStack Developer en formación.

---

##  Objetivo del proyecto

Construir mi portfolio real desde cero mientras documento absolutamente todo el proceso.

Este mismo proyecto irá evolucionando versión a versión, reflejando lo que voy aprendiendo en frontend, backend, UI/UX y buenas prácticas.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- Bootstrap 5 (solo lo esencial)  
- FontAwesome (iconos)
